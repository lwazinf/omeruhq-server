# OmeruWA — WhatsApp Bot

The conversational commerce layer. Customers browse stores, add to cart, and place orders through WhatsApp.

## What it handles
- Customer sessions (`UserSession` — mode, cart_json, active_merchant_id)
- Order placement via conversation
- Broadcast delivery from merchants
- Booking confirmations

## Key data
- `UserSession.mode`: CUSTOMER / MERCHANT
- `UserSession.cart_json`: live cart state
- Order lifecycle starts here → visible in OmeruHQ
