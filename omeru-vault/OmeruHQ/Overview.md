# OmeruHQ — Merchant Portal

The merchant-facing dashboard. Merchants manage orders, products, bookings, customers, analytics, and payouts.

## Feature set
- Phase 1: Login, Dashboard, Orders (Kanban), Products
- Phase 2: Services, Bookings, Broadcast, Team, Settings
- Phase 3: Analytics, Customers, Payments/Payouts, Reviews (placeholder)
